Yij - поток из  i вершины по j графу

Cij - норма стоимость потока из  i вершины по j графу

R+ множество вещественных положительных чисел

si - потоки для вершин из множества S (вершины изначальные)
ti - потоки для вершин из множества T (вершины target)

argmin - возвращает значение аргумента, при котором достигается минимум функции

Xi - локальные координаты каждого изображения
